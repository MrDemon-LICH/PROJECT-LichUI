About LichUI

LichUI is a custom firmware currently in development for the Samsung Galaxy A54 5G. Built on the latest version of Samsung's One UI, it adds exclusive features and optimizations to deliver the best possible user experience.

Developer

Hi, I'm the developer of LichUI. The project was originally born on my Galaxy A51 device in order to have my own custom ROM and then share it with others. Over time, the project grew and I changed my device to the Galaxy A54, on which the project is currently based.

Development team

LichUI is currently made up of a development team in charge of testing, module development, design, among others; without them the project would not advance so quickly.
